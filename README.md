# Northwind Demo

Playground for web technologies using a PostgreSQL version of the Northwind database

## Database

### Start

```shell-script
npm run db-start
```

### Stop

```shell-script
npm run db-stop
```

## API

View the API's README [here](./api/README.md).

### Start

```shell-script
npm run api-start
```

### Start in Watch Mode

```shell-script
npm run api-watch
```
